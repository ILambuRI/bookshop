Shop:

authors:

[GET] bookshop/client/api/shop/authors/
Get all the authors from the table. (grouping by id)

[GET] bookshop/client/api/shop/authors/(id)
Get full information about the author by id

genres:

[GET] bookshop/client/api/shop/genres/
Get all the genres from the table. (grouping by id)

[GET] bookshop/client/api/shop/genres/(id)
Get full information about the genre by id

books:

[GET] bookshop/client/api/shop/books/
Get all the books from the table

[GET] bookshop/client/api/shop/books/id(books)/id(authors)/id(genres)
Get full information about the books where:
/id(books).
/false(null)/id(authors).
/false(null)/false(null)/id(genres).
Return array.

payment:

[GET] bookshop/client/api/shop/payment/
Get all payments from the table. (grouping by id)

status:

[GET] bookshop/client/api/shop/status/
Get all status from the table. (grouping by id)



User:

users:

[GET] bookshop/client/api/user/users/(login)/(hash)
/null(or false)/hash - check hash in table or hash lifetime.
/login - check if there is a login in the table.
Return 200 or 400+.

[POST] bookshop/client/api/user/users/
Registration - write a new user in table.
Input: 3 < login > 20 - unique | 5 < password > 32 | 6 < phone > 20
Return hash for autologin or 400+.

[PUT] bookshop/client/api/user/users/
Login - user authorization, if it is in the table. Return new hash.
Input: 3 < login > 20 - unique | 5 < password > 32.
Return new hash or 400+.

[DELETE] bookshop/client/api/user/users/(hash)
Logout - removing (updating) a hash in table.
Return 200 or 400+.

cart:

[GET] bookshop/client/api/user/cart/(hash)
Getting all the books from the cart by user hash
/hash - input.
Return array or 400+.

[POST] bookshop/client/api/user/cart/
Adding a book to the cart.
hash | id(book) | count - input.
Return 200 or 400+.

[PUT] bookshop/client/api/user/cart/
Changing the number of books in the cart.
hash | id(book) | count - input.
Return 200 or 400+.

[DELETE] bookshop/client/api/user/cart/(hash)/(id)
Removing a book from a cart.
/hash/id(book) - input.
Return 200 or 400+.

orders:

[GET] bookshop/client/api/user/orders/(hash)
Receive all user orders through a unique users hash.
/hash - input.
Return array or 400+.

[POST] bookshop/client/api/user/orders/
Record a new order in the table orders(and info_order) and clear user cart.
hash | id(payment) - input.
Return order ID or 400+.

hash(admin) | id(authors) | name(new author name) - input.
Return 200 or 400+.

[DELETE] bookshop/client/api/admin/authors/(hash)/(id)

/hash(admin)/id(author) - input.
Return 200 or 400+.

Receive all orders for admin.

Set a new status for the order.

users:

[GET] bookshop/client/api/admin/users/(hash)/(id)
/hash(admin) - get all users.
/hash(admin)/id(user) - get the user by ID.
Return array or 400+.

[POST] bookshop/client/api/admin/users/
Write a new user in table.
hash(admin) | login | password | phone | idDiscount | admin(1 or 0) | active(1 or 0)- input.
Return 200 or 400+.

[PUT] bookshop/client/api/admin/users/
Update user data.
hash(admin) | id(users) | login | password | phone | idDiscount | admin(1 or 0) | active(1 or 0)- input.
Return 200 or 400+.

books:

[POST] bookshop/client/api/admin/books/
Adding a new book to the table.
hash(admin) | name(books) | description | pubyear(int) | price(int) | idDiscount
authorsId(1,2,3 etc.) | genresId(1,2,3 etc.) - input.
Return id(new book) or 400+.

[PUT] bookshop/client/api/admin/books/
Update book in the table.
hash(admin) | id(book) | name(books) | description | pubyear(int) | price(int) | idDiscount
authorsId(1,2,3 etc.) | genresId(1,2,3 etc.) - input.
Return 200 or 400+.
